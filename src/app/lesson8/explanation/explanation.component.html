Lesson 8: Lists, Part 2

Introduction to Lists

A list in functional programming is a sequence of elements, all of the same type. For example: [2, 18, 9, 10] The component type can be any simple or compound type, e.g., [Int]

Definition

  • A list is either empty [], or consists of a head (a single element) and a tail (another list).
  • The head and tail are connected using the cons operator ::

Example: [2, 18, 9, 10] is equivalent to 2::(18::(9::(10::[])))

Operators and Functions

Function Signature Examples Application
head [a] -> a head([2, 18, 9, 0]) = 2 head(x::xs) = x
tail [a] -> [a] tail([2, 18, 9, 0]) = [18, 9, 0] tail(x::xs) = xs
length [a] -> Int length([2, 18, 9, 0]) = 4 length([]) = 0

Structural Induction

Structural induction is used to prove propositions for recursively defined structures like lists or trees. The base case is typically the empty list [], and the induction case involves non-empty lists x::xs

Example: Length Function

  • len([]) = 0 (base case)
  • len(x::xs) = 1 + len(xs) (induction case)

Example: Sum Function

  • sum([]) = 0 (base case)
  • sum(x::xs) = x + sum(xs) (induction case)

Functions and Descriptions
